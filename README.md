# Home_Server_Docs
Document my home server(Arch GNU/Linux BTW), the servers, programs and configs i will use. What the Usage of that server, file server mostly.

# Why Arch? (BTW)
To be simple and challenging journey for studing linux and getting expertise and well controlled on my server.

# About The Server 
- Its an old computer with 3rd gen intel cpu and it's integrated gpu, and 6Gb of memory.
- The network interface is a "USB Wi-Fi adapter", not ethernet connections.
- With 2 HDDs,75 as it's main disk(OS disk) and 500Gb as storage disk.
- Arch Based with GRUB(x86_64,EFI) as it bootloader and systemd as it init system.
- Just for personal use nothing like a professional production server or a fancy documentation.
- It will be a file server and a communication hub in my network, allowing all devices on it to communicate with each other.
- I also have some plaing to make a proxy server on it, but may not doing it.

# Rules 
- Not to use external pkgs out from the pacman offical repos exept thoese i have built my self
